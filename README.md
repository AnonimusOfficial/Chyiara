# Chyiara

A biblioteca para governar todos eles. A mãe da Kiyara.

## Filosofia

**Chyiara** não é apenas uma biblioteca, é um conceito. Inspirada na filosofia de que a verdadeira maestria reside na discrição, a Chyiara não expõe sua natureza interna de forma explícita, mantendo uma aura de mistério. Ela é a força silenciosa que permite que seus "filhos" (os bots) brilhem e interajam com o mundo digital.

## Propósito

O objetivo principal da Chyiara é fornecer uma interface robusta e flexível para a comunicação via WebSocket, começando com o WhatsApp. Ela serve como a fundação para a criação de bots e automações, permitindo a exploração e a engenharia reversa de protocolos complexos de forma estruturada.

## Status

O projeto está em sua fase inicial. A comunicação base via WebSocket foi estabelecida com sucesso.
